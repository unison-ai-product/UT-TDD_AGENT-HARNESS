# Agent Cost Design

AIエージェント構築の着手前に、コスト予算と暴走防止ガードレールを確定させるオーケストレーションスキル。

設計・実装に着手する前にこのスキルを通す。「コスト見積もり不要」「予算は気にしない」と明示された場合のみスキップ可。

## トリガー

- AI エージェント / LLM サブエージェント / マルチエージェントの構築・改修タスクを受領
- 生成フローに新しい LLM 呼び出しを追加する
- 既存エージェントのコスト超過 / 予算消費 80% 到達
- フォールバック / リトライ / ベンダー切替の設計判断

## 設計思想

- ガードレールは AI に判断させない。外側のラッパー層でハードリミットとして実装する
- 上振れ係数は 1.2 固定。これを超える見積もりが出る時点で設計を見直す
- 予算消費 80% 到達で追加予算見積もりを提示し、判断を仰ぐ
- モデル単価は揮発するため、スキル実行ごとに必ず Web 検索で最新値を取得する
- 1 回実行コストは階層別ではなく、生成フロー全体の総和で算出する
- フォールバックは原則禁止。例外はインフラ起因の API リトライ・ベンダー切り替えのみ
- マルチベンダー対応は設計の最低要件

---

## 実行順序

以下の順序で各 Phase を確定させる。各 Phase 完了まで次に進まない。

### Phase 0: 前提情報の確認

以下を確認する。未指定の項目は推奨デフォルトを提示して確認する。

- 構築予算の上限額（円または USD）
- エージェントの想定タスク内容
- 想定実行頻度
- テスト方針（実 API / モック中心 / リリース前のみ実 API）

### Phase 1-pre: 設計原則の確認と生成フロー設計

以下を順に確認する。

1. マルチベンダー対応設計の要件確認
2. フォールバック禁止原則の確認
3. API リトライ設計の組み込み
4. 生成フロー全体の洗い出し

すべての原則を満たす設計になっていることを確認してから Phase 1 に進む。

### Phase 1: 生成コスト見積もり

Web 検索で主要プロバイダの最新単価・ベンチマーク・エフォート設定を取得し、各呼び出しにモデルを割り当て、1 回実行コスト（生成フロー全体の総和）を算出する。

### Phase 2: テスト予算設計

単体テスト・E2E テストのコスト基準を確定する。

### Phase 3: 即時停止ガード値とガードレール実装方針

E2E 上限コスト（×1.2）と、ラッパー層で実装すべきハードリミットを確定する。

### Phase 4: 完成予算の総合算出

単体テスト・E2E テスト・開発中試行バッファを合算し、完成予算を算出する。Phase 0 で確認した予算上限と照合する。

### Phase 5: 予算監視と追加予算申請

80% 到達時の追加予算申請テンプレート、100% 到達時の自動停止挙動を実装方針として確定する。

## チェックリスト

スキル実行時、以下を順番に確認する。

- [ ] Phase 0: 予算上限・タスク内容・実行頻度・テスト方針を確認した
- [ ] Phase 1-pre: マルチベンダー対応設計になっている
- [ ] Phase 1-pre: フォールバック禁止原則を満たしている
- [ ] Phase 1-pre: API リトライ設計を組み込んでいる
- [ ] Phase 1-pre: 生成フロー全体の全 LLM 呼び出しを洗い出した
- [ ] Phase 1: Web 検索で最新単価・ベンチマーク・エフォート設定を取得した
- [ ] Phase 1: 1 回実行コスト（生成フロー全体の総和）を算出した
- [ ] Phase 2: 単体テスト・E2E テストのコスト基準を提示した
- [ ] Phase 3: E2E 上限コスト（×1.2）とガードレール実装方針を確定した
- [ ] Phase 4: 完成予算を算出し予算上限と照合した
- [ ] Phase 5: 80%/100% の挙動を実装方針として明示した

すべて YES になるまで設計・実装フェーズに進まない。

## UT-TDD との関連

このスキルはエージェント構築タスク受領時の前段必須ゲートである。
標準フロー: **agent-cost-design（前段）→ agent-design（個別構造）→ agent-teams（協調）**
コストガードを通さずに structural design へ進まない。

UT-TDD での委譲: `ut-tdd codex --role se --task "エージェント設計 ..."` 着手前にこのスキルを通す。
