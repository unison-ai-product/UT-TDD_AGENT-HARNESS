# L13 デプロイ後検証・実環境運用

## 工程の位置づけ

| 項目 | 内容 |
|---|---|
| 区分 | V字 右腕（運用フェーズ） |
| 入力 | L12 デプロイ |
| 出力 | L14 への入力 |
| 対応する設計 | — |

## この工程の PLAN

PLAN は機能（ドキュメント）単位で起票し、工程表（作成手順＋進捗）と実装計画を内蔵する。

### `L13-デプロイ後検証plan`
- 本番動作確認
- 初期監視

### `L13-実環境運用plan`
- 運用開始
- 運用手順

## PLAN が内蔵するもの

- **工程表**: そのドキュメントを完成させる手順（例: 参考調査の Web 検索 → 既存資料・ヒアリング整理 → ドラフト → TL レビュー → 確定）と各手順の進捗
- **実装計画**: 記載項目をどう埋めるかの計画
