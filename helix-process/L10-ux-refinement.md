# L10 フロントデザインUX・ビジネスデザイン磨き上げ

## 工程の位置づけ

| 項目 | 内容 |
|---|---|
| 区分 | V字 右腕（検証フェーズ） |
| 入力 | L9 総合テスト・L2 画面設計 |
| 出力 | L11 への入力 |
| 対応する設計 | L2 画面設計・フロントUI / ワイヤーモック |

## この工程の PLAN

PLAN は機能（ドキュメント）単位で起票し、工程表（作成手順＋進捗）と実装計画を内蔵する。

### `L10-UX磨き上げplan`
- UI / UX 改善
- 操作性向上

### `L10-ビジネスデザインplan`
- ビジネス価値の磨き上げ

## PLAN が内蔵するもの

- **工程表**: そのドキュメントを完成させる手順（例: 参考調査の Web 検索 → 既存資料・ヒアリング整理 → ドラフト → TL レビュー → 確定）と各手順の進捗
- **実装計画**: 記載項目をどう埋めるかの計画
