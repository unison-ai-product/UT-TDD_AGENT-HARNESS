# L8 結合テスト

## 工程の位置づけ

| 項目 | 内容 |
|---|---|
| 区分 | V字 右腕（検証フェーズ） |
| 入力 | L7 実装・L5 結合テスト設計 |
| 出力 | L9 総合テスト への入力 |
| 対応する設計 | L5 詳細設計 / 結合テスト設計 |

## この工程の PLAN

PLAN は機能（ドキュメント）単位で起票し、工程表（作成手順＋進捗）と実装計画を内蔵する。

### `L8-結合テストplan`
- モジュール間結合確認
- インターフェース結合確認

### `L8-依存関係解消plan`
- 依存不整合の検出
- 解消対応

## PLAN が内蔵するもの

- **工程表**: そのドキュメントを完成させる手順（例: 参考調査の Web 検索 → 既存資料・ヒアリング整理 → ドラフト → TL レビュー → 確定）と各手順の進捗
- **実装計画**: 記載項目をどう埋めるかの計画
